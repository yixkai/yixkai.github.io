Lyra 数据中台

数据全生命周期一站式开发运营平台
提供数据集成、数据开发、数据治理、数据服务、数据可视化等功能
轻
一键部署，按需部署，支持docker容器化部署
易
拖拽式操作，自助式分析。无需编程也能搞定数据处理与分析
全
报告、大屏、数据门户、自助取数、复杂报表等产品功能应有尽有
便
可视化的任务运维中心，监控所有任务的运行，支持配置各类报警

一站式大数据开发及管理平台

提供数据集成，数据开发，数据治理及数据服务等全方位的产品服务

数据集成
整合各类数据源
可视化配置，支持批量数据迁移、实时数据集成和数据库实时同步，全向导式配置和管理
元数据管理
数据血缘和数据全景可视化
自动化元数据中心构建，支持采集、解析、管理数据构建过程及结果的所有元数据
智能数据开发
建模研发自动化
自顶向下可视化构建数据架构、规范定义数据对象并自动生成代码，设计即研发的提效同时确保数据标准、规范和统一
数据可视化
高效专业的展示数据
拖拽即可完成样式和数据配置，无需编程就能轻松搭建数据大屏
实时流计算
高性能、低延迟
基于 Apache Flink 构建的高性能、一站式实时大数据处理方案，适用于流式数据处理场景，通过标准 SQL + UDF服务，提供低门槛，专业化的实时计算 IDE

核心能力

规范定义
依托阿里的OneData方法论，对维度、业务过程、原子指标、衍生指标等进行定义，保证数据规范性，严格定义指标规范，规避指标二义性；
海量数据同步
可通过数据传输产品完成丰富数据源之间的互相传输；同时支持离线全量、增量实时等多种同步方式。批量同步采用Spark引擎，实时同步采用Flink计算引擎。
智能数据研发
支持配置逻辑数据模型，包括维度模型、事实模型和汇总模型，模型定义提交后，逻辑模型通过智能计算引擎自动生成物理模型和调度任务。
便捷的数据开发
深度集成DolphinScheduler，提供了界面化、智能高效的大数据数据开发体验，并提供全托管的调度能力。
数据质量监控
自定义规则，从完整性、有效性、及时性、一致性、准确性、唯一性六个维度，结合质量作业、对账作业进行单列、跨列、跨行和跨表的分析。
实时流计算
基于 Apache Flink 构建的高性能、一站式实时大数据处理方案，适用于流式数据处理场景，通过标准 SQL + UDF服务，提供低门槛，专业化的实时计算 IDE
